## Easy Quickstart

```bash
npm install -g gatsby-cli
```
##### Create a new site
```bash
gatsby new gatsby-site
```
##### Change to the site folder
```bash
cd gatsby-site
```
##### Start dev server
```bash
gatsby develop
```

<Notes>
Easy CLI just like most programming languages
</Notes>

##### install
```bash
npm install --save gatsby-source-filesystem
```
##### Add plugin gatsby-config.js
```
module.exports = {
  siteMetadata: {
    title: `blah`,
  },
  plugins: [
    {
      resolve: `gatsby-source-filesystem`,
      options: {
        name: `src`,
        path: `${__dirname}/src/`,
      },
    }
  ],
}
```
